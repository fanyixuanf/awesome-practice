## 安装
-   安装 kubectl
-   Mac brew 换源 [brew 换源](https://github.com/fanyixuanf/Blog/blob/master/mac/brew.md)
-   Mac 安装 `brew install kubectl` 或 `brew install kubernetes-cli`
-   测试安装环境 `kubectl version --client`
-   配置shell 自动补全
	-	Bash
	- 	Zsh
